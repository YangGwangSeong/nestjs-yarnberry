# Library

1. [cross-env](https://www.npmjs.com/package/cross-env)

```bash
$ yarn add cross-env
```

> 모든 운영체제에서 동일한 방법으로 환경 변수를 변경할 수 있게 된다.

2. [Configuration](https://www.npmjs.com/package/@nestjs/config)

```bash
$ yarn add @nestjs/config
```

> 환경 변수 설정을 위해 설치합니다.
